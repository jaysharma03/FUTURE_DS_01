# 📊 Superstore Sales Dashboard

🚀 An interactive Power BI dashboard delivering actionable business insights from retail sales data.

---

## 🧭 Overview
This project is a business sales dashboard built using Power BI. It focuses on analyzing retail performance by exploring sales trends, product performance, customer segments, and regional profitability.
The dashboard transforms raw data into actionable insights, enabling businesses to make informed, data-driven decisions and identify growth opportunities. 

# 📂 Dataset
- Source:  [Superstore Dataset](https://raw.githubusercontent.com/jaysharma03/FUTURE_DS_03/main/SuperStore_Sales_%20DataSet.csv)
- Format: CSV  

### Key Attributes:
- Order ID, Product Name  
- Category, Sub-Category  
- Sales, Profit, Quantity  
- Order Date, Ship Date  
- Region, Customer Segment  

---

## 🛠️ Tech Stack
- Power BI (Visualization & Dashboard)  
- DAX (Calculations & KPIs)  
- Excel (Optional – Data Cleaning)  

---

## ✨ Key Features

### KPI Metrics
- Total Sales  
- Total Profit  
- Total Orders  
- Profit Margin (%)  

### Visualizations
- Sales Trend (Line Chart)  
- Top 10 Products by Sales  
- Sales & Profit by Category  
- Profit by Region  

### Interactivity
- Dynamic filters (Top N products)  
- Slicers for Category and Region  
- Interactive dashboard exploration  

---

## 📊 Key Insights

- Total Sales exceeded $2M with 175K profit  
- Sales peak during November–December (seasonal trend)  
- Technology category generates highest profit  
- West region contributes the most to profitability  
- Some high-selling products have low profit margins  

---

## ▶️ Usage
1. Download the .pbix file  
2. Open in Power BI Desktop  
3. Use slicers and filters to explore insights  
4. Export reports if needed
