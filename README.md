# 📊 Superstore Sales Dashboard

🚀 An interactive Power BI dashboard delivering actionable business insights from retail sales data.

---

## 🧭 Overview
This project showcases a dynamic and interactive sales dashboard built using Power BI to analyze a retail superstore’s performance.

It provides a 360° view of business operations, covering:
- Sales trends  
- Profitability  
- Product performance  
- Regional insights  

The dashboard helps stakeholders make data-driven decisions by identifying growth opportunities and performance gaps.

---

## 📂 Dataset
- Source: Superstore Dataset  
- Format: CSV  

### Key Attributes:
- Order ID, Product Name  
- Category, Sub-Category  
- Sales, Profit, Quantity  
- Order Date, Ship Date  
- Region, Customer Segment

## ✨ Key Features

### KPI Metrics
- Total Sales  
- Total Profit  
- Total Orders  
- Profit Margin (%)  

### Visualizations
- Sales Trend (Line Chart)  
- Top 10 Products by Sales  
- Sales & Profit by Category  
- Profit by Region  

### Interactivity
- Dynamic filters (Top N products)  
- Slicers for Category and Region  
- Interactive dashboard exploration  

---

## 📊 Key Insights

- Total Sales exceeded $2M with 175K profit  
- Sales peak during November–December (seasonal trend)  
- Technology category generates highest profit  
- West region contributes the most to profitability  
- Some high-selling products have low profit margins

## ▶️ Usage
- Download the `.pbix` file  
- Open it in Power BI Desktop  
- Use slicers and filters to explore insights  
- Analyze trends across time, region, and products
